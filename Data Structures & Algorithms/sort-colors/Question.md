# 🔴⚪🔵 Sort Colors
**Difficulty:** Medium  
**Source:** NeetCode / LeetCode  
---
## 📌 Problem
You are given an array `nums` where each element represents a color:
- `0` → Red 🔴  
- `1` → White ⚪  
- `2` → Blue 🔵  

Your task is to **sort the array in place** so that elements are grouped in the order:
0 → 1 → 2
> ⚠️ You must **not use any built-in** sort functions.

---

## 🧪 Examples

### Example 1
```bash
Input: nums = [1,0,1,2]
Output: [0,1,1,2]
```
### Example 2
 ```bash
Input: nums = [2,1,0]
Output: [0,1,2]
```
🔒 Constraints
```bash
1 <= nums.length <= 300
0 <= nums[i] <= 2
```
